# 초기 실험 계획

## 목적

코엑스 내부 전체 위치추정으로 확장하기 전에, 제한된 구역에서 이미지 기반 위치추정의 정확도와 실패 조건을 검증합니다.

## 입력 데이터

- iPhone 15 Pro Max로 촬영한 사용자 관점 이미지
- Polycam으로 수집한 공간 스캔
- 스캔에서 추출한 키프레임 이미지와 카메라 자세
- 스캔 또는 재구성에서 얻은 깊이, 메시, 포인트 클라우드
- 코엑스 평면도 좌표계와의 정렬 기준점

## 베이스라인

1. 전역 임베딩 검색
   - DINOv2-small을 기본값으로 사용합니다.
   - 신규 이미지 1장에 대해 후보 keyframe top-K를 반환합니다.

2. 로컬 특징 검증
   - XFeat와 LightGlue 조합을 우선 검토합니다.
   - 후보 keyframe과 query 이미지 간의 기하 일관성을 평가합니다.

3. 2D-3D 자세 추정
   - 충분한 2D-3D 대응점이 있을 때 PnP 기반 위치를 계산합니다.
   - 대응점이 부족하면 이미지 검색 기반 후보 위치만 반환합니다.

## 평가 지표

- top-1 위치 오차
- top-5 후보 안에 정답 구역이 포함되는 비율
- 1m 이내 성공률
- 3m 이내 성공률
- 5m 이내 성공률
- 실패 또는 불확실 판정의 정확도
- query 1장당 평균 처리 시간

## 목표 기준

초기 파일럿 기준은 다음과 같습니다.

- 3m 이내 성공률 70% 이상
- 5m 이내 성공률 85% 이상
- 불확실한 입력에 대해 낮은 신뢰도를 반환
- query 1장당 후보 검색 100ms 이하
- 정밀 검증을 포함한 전체 응답 2초 이내

## 실패 케이스

- 사람이 많이 가려진 이미지
- 반복 구조가 많은 복도
- 조명이 강하게 바뀐 장소
- 매장 간판이나 장식이 바뀐 장소
- 스캔 시점과 사용자 촬영 시점의 이동 동선 차이가 큰 경우
- 유리, 반사, 대형 광고판처럼 시각 특징이 불안정한 구역
