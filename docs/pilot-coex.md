# 코엑스 파일럿 수집 계획

## 범위

최종 목표는 코엑스 내부 전체입니다. 첫 파일럿은 지하층의 제한된 구간에서 시작해 데이터 수집, 좌표 정렬, 검색 정확도, 실패 조건을 먼저 검증합니다.

## 수집 장비

- iPhone 15 Pro Max
- Polycam Space 또는 LiDAR 스캔
- 별도 query 촬영용 기본 카메라 앱

## 구역 단위

초기 파일럿은 다음처럼 구역을 나눕니다.

```text
B1-STARFIELD
B1-MALL-CORRIDOR-A
B1-EXHIBITION-LINK-A
B2-PARKING-LINK-A
```

각 구역은 독립적으로 스캔하고, 나중에 공통 평면도 좌표계로 합칩니다.

## 수집 절차

1. 구역 입구, 교차점, 에스컬레이터, 엘리베이터, 안내판처럼 위치 식별이 쉬운 기준점을 정합니다.
2. Polycam으로 구역을 왕복 스캔합니다.
3. 스캔 결과에서 키프레임 이미지, 카메라 자세, 메시 또는 포인트 클라우드를 추출합니다.
4. 같은 구역에서 query 평가용 사진을 별도로 촬영합니다.
5. query 사진에는 정답 좌표, 층, 구역, 촬영 방향, 조명 상태를 기록합니다.
6. 사람 얼굴, 결제 화면, 민감한 매장 내부 정보는 비식별화하거나 제외합니다.

## 촬영 밀도

초기 기준은 다음과 같습니다.

```text
키프레임 간격: 1-2m
query 사진: 주요 지점당 4방향
반복 촬영: 평일/주말 또는 혼잡/비혼잡 최소 2회
```

## 성공 기준

- top-5 후보 안에 같은 구역의 정답 위치가 들어오는 비율
- 1m, 3m, 5m 이내 위치 성공률
- 반복 구조 구간에서 오탐률
- 실패 또는 불확실 판정 비율
- query 1장당 응답 시간

초기 MVP 기준은 `3m 이내 70%`, `5m 이내 85%`, 정밀 검증 포함 `2초 이내`를 목표로 둡니다.
