# 데이터 정책

## 원칙

`path-finder`는 공개 레포이지만, 위치추정에 쓰이는 원본 공간 데이터는 공개하지 않는 것을 기본값으로 합니다. 코엑스 내부 스캔에는 사람, 매장, 보안 설비, 상업 정보가 함께 담길 수 있기 때문입니다.

## 공개 가능한 항목

- 코드
- 실험 스크립트
- 모델 평가 요약
- 익명화된 소형 샘플
- 합성 데이터 또는 외부 공개 데이터셋 기반 예제
- 데이터 스키마와 처리 절차

## 공개하지 않는 항목

- 원본 Polycam 스캔
- 원본 키프레임 이미지
- 얼굴이나 개인정보가 포함된 이미지
- 매장 내부 민감 정보
- 보안 설비 위치가 드러나는 고해상도 지도
- 운영용 위치 데이터베이스

## 수집 시 주의사항

- 현장 촬영 정책과 시설 관리 주체의 허가 조건을 확인합니다.
- 사람 얼굴과 개인정보가 포함되지 않도록 촬영 방향과 시간을 조정합니다.
- 공개 또는 공유 전에 자동 마스킹과 수동 검수를 모두 수행합니다.
- 실험용 데이터와 운영용 데이터를 분리합니다.
- 삭제 요청이 들어올 수 있는 데이터를 추적 가능하게 관리합니다.

## 저장 정책

- 공개 레포에는 원본 데이터 대신 자리 표시자와 문서만 둡니다.
- 원본 데이터는 암호화된 비공개 스토리지에 보관합니다.
- 데이터 버전, 수집 날짜, 구역, 장비, 처리 파이프라인 버전을 함께 기록합니다.
- 모델 학습 또는 평가에 사용한 데이터 목록은 재현 가능한 manifest로 관리합니다.
